# MK Image Converter Expert - README

## 概要 / Overview
このソフトは、複数の画像を直感的な操作で一括変換・加工するためのツールです。
This software is a tool for batch converting and processing multiple images.

## 主な機能 / Key Features
- ドラッグ＆ドロップによる画像追加 (Drag & Drop support)
- 一括リサイズ (Batch resizing)
- 透かし（ウォーターマーク）合成 (Watermarking)
- 多言語対応 (Multilingual: JP/EN)
- プリセット保存機能 (Save settings as presets)

## 使い方 / How to Use
1. 画像をエリアにドラッグ＆ドロップします。
2. サイズや透かし、保存名のルールを設定します。
3. 「一括変換スタート」をクリックして保存先を選びます。
