# 📸 MK Image Converter Expert

複数画像の一括リサイズ、形式変換、透かし合成をシンプルに。

## ✨ 主な機能
- **ドラッグ&ドロップ**: 複数の画像をまとめて追加
- **一括リサイズ**: 指定サイズへ高速変換
- **透かし合成**: テキストを好きな位置・サイズで挿入
- **多言語対応**: 日本語と英語の切り替えが可能
